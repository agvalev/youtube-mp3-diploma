🎵 yt-downloader-gui

yt-downloader-gui is a professional cross-platform desktop application for downloading and managing multimedia content from YouTube through a modern graphical interface.

The project is developed in Python using the PyQt6 framework and integrates powerful tools such as yt-dlp for media downloading and ffmpeg for audio and video processing.

The application focuses on strong software architecture, usability, security and background processing, making it suitable for real-world use and academic diploma defense.

🎯 Project Goals

The goal of the project is to demonstrate how a multimedia desktop system can be built by combining:

  • Modern graphical user interface
  • Powerful media extraction backend
  • Threaded background processing
  • Secure URL validation
  • Clean modular software architecture

All technical complexity is hidden from the user. Heavy operations run safely in the background while the user interacts with a simple interface.

🧠 How the Application Works

  1. User enters a YouTube URL
  2. The application validates the URL
  3. The system detects the link type automatically
  4. Metadata is extracted without downloading media
  5. User selects desired items
  6. Downloads are added to a managed queue
  7. Downloads run in background threads
  8. Progress is displayed in real time
  9. Files are processed and saved locally

The user interface never freezes even during long downloads.

⚙ Backend Processing

yt-dlp Responsibilities

  • URL validation
  • Metadata extraction
  • Stream resolution
  • Playlist and channel handling
  • Download execution

ffmpeg Responsibilities

  • Audio/video merging
  • MP3 conversion
  • Bitrate and quality control
  • Encoding parameter management

🔐 Security

Security is an important design principle.

  • Only trusted YouTube domains are accepted
  • Invalid URLs are rejected
  • yt-dlp performs additional validation
  • No execution of external files
