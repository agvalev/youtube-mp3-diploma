Anatoli Valev Project

A professional, cross-platform desktop application for downloading and managing multimedia content from YouTube — built with a modern graphical interface, clean modular architecture, and robust background processing.

Powered by yt-dlp for media extraction and ffmpeg for audio/video processing. Suitable for real-world use and academic diploma defense.

✓ Python 3.10+ ✓ PyQt6 ⚙ yt-dlp ⚙ ffmpeg 🌐 HTML · CSS · JS 🖥 Windows · Linux · macOS
🧠

How the Application Works

The app acts as a controller layer between the user and two specialized media engines — yt-dlp for discovery and downloading, and ffmpeg for processing. The UI never freezes, even during long downloads.

1

User enters a YouTube URL

2

The application validates the URL against trusted domains

3

Link type is detected automatically (video / playlist / channel / shorts)

4

Metadata is extracted without downloading media

5

User selects desired items (if playlist or channel)

6

Downloads are added to a managed queue

7

Media is downloaded in background threads — UI stays responsive

8

Progress is parsed and displayed in real time

9

Files are processed via ffmpeg and saved locally

Main Features

🎯 Download Modes

Single video, MP3 extraction, playlist, channel videos, channel MP3, and YouTube Shorts.

🎚 Quality Control

Video: best available or fixed resolution (8K → 1080p). Audio: MP3 up to 320 kbps.

🔑 Authentication

Cookie-based login for age-restricted, members-only, and private videos via cookies.txt.

🔄 Auto-Updater

GitHub version check for yt-dlp, automatic binary replacement — no manual updates needed.

📊 Live Feedback

Progress bar, real-time activity log, console output parsing, and automatic scrolling view.

🌐 Built-in Web UI

The help system and UI components use embedded HTML, CSS, and JavaScript — rendered inside the app via PyQt6's WebEngine.

🧩 Modular Design

Clean separation of UI, download logic, auth, and update logic across dedicated modules.

Security

🛡

Security is a core design principle. Only trusted YouTube domains are accepted. All URLs are validated before any processing begins. No execution of untrusted external binaries.

  • Only youtube.com, youtu.be, music.youtube.com are accepted
  • Invalid or unknown URLs are rejected immediately
  • yt-dlp performs additional internal validation
  • Cookies (if used) are never stored permanently or transmitted externally
  • No user credentials are logged or persisted

Installation

Requires Python 3.10+. On Windows, yt-dlp and ffmpeg are bundled in bin/. On Linux/macOS, ffmpeg must be installed and available in the system PATH.

bash — install & run
# Clone the repository
git clone https://github.com/uikraft-hub/yt-downloader-gui.git

# Navigate to the project directory
cd yt-downloader-gui

# Install Python dependencies
pip install -r requirements.txt

# Run the application
python src/main.py

Configuration

Cookie Authentication (Optional)

To download age-restricted, members-only, or private videos, provide a cookies.txt file exported from your browser.

  • Export cookies using a supported browser extension
  • Place cookies.txt in the project root, or select it via the UI
  • Cookies are passed directly to yt-dlp — never stored or transmitted by the app
